## 摘要
即时重建是机器人与自主导航中许多应用的关键需求。变分贝叶斯高斯泼溅（VBGS）利用坐标上升变分推断（CAVI）实现了无需回放缓冲区的连续学习，但其对每个观测点的逐帧迭代使得在严格的内存和延迟要求下无法实时使用。我们提出了ImprovedVBGS，一种用于即时连续重建的加速框架。这主要通过以下两点实现：（i）空间截断变分推断，以及（ii）改进的重新分配，该分配利用前向传播、截断并消除了浪费的动态重编译。在NeRF合成数据集上，我们在RTX 3070 Ti上将平均每帧延迟从约84.0秒降至约0.050秒，实现了1680倍的加速，同时保持了重建质量。

## Abstract
On-the-fly reconstruction is a key requirement for many applications in robotics and autonomous navigation. Variational Bayes Gaussian Splatting (VBGS) enables continual learning without replay buffers using Coordinate Ascent Variational Inference (CAVI), but its per-frame iterations over all observed points make it too slow for real-time use with strict memory and latency requirements. We present ImprovedVBGS, an accelerated framework for on-the-fly continual reconstruction. This is achieved primarily through (i) spatially truncated variational inference, and (ii) improved reassignment that uses forwarding, truncation and eliminates wasteful dynamic recompilation. On the NeRF synthetic dataset, we reduce mean per-frame latency from ~84.0 s to ~0.050 s on an RTX 3070 Ti, a 1680x speed-up while maintaining reconstruction quality.

### 1. 论文的核心问题与整体含义
- **研究动机与背景**：在机器人和自主导航等应用中，**即时在线重建**是核心需求。变分贝叶斯高斯泼溅（VBGS）通过坐标上升变分推断（CAVI）实现了无需经验回放的连续学习，从根本上避免了“灾难性遗忘”问题。
- **核心问题**：尽管VBGS避免了遗忘，但其原始实现存在严重缺陷。它需要在每一帧对所有已观测的数据点（点云）执行迭代，导致计算量随着场景规模线性增长。这造成了极高的延迟，例如在NeRF合成数据集上每帧处理时间高达**84.0秒**，完全无法满足对延迟和内存有严格限制的实时应用需求。

### 2. 论文提出的方法论
- **核心思想**：在保持VBGS无需回放缓冲区的连续学习能力前提下，通过**空间局部性**和**计算优化**来大幅减少每帧的冗余计算，从而将训练加速到实时水平。
- **关键技术细节**：
    1.  **空间截断变分E步（Spatially Truncated Variational E-step）**：
        - 不再计算每个数据点与**所有**高斯分量的关联度，而是利用KD树为每个数据点寻找其在空间上**最近的C个**高斯分量。
        - 仅在这C个候选分量上计算对数评分和软分配责任（responsibility），将计算复杂度从`O(nK)`（n为点数，K为分量数）降低到`O(nC)`，其中`C << K`。
    2.  **改进的重分配策略（Improved Reassignment）**：
        - **前向传播与截断**：将重分配步骤的执行顺序调整到变分EM步骤之后，并直接复用E步中已经计算好的、低ELBO（证据下界）分数的点作为重分配的目标。这不仅利用了截断带来的效率，还通过调整步骤顺序进一步减少了计算。
        - **静态张量填充**：由于每帧需要重分配的分量数`n_reassign`是动态变化的，这会导致JAX（一种自动微分框架）动态编译，产生额外开销。此方法将所有帧的重分配索引数组大小固定为一个静态编译时的最大值`n_max`，从而**完全消除了动态重编译**的开销。
