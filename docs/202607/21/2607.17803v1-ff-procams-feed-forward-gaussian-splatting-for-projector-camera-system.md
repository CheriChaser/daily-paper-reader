## 摘要
投影仪-相机（ProCams）系统通过结构光实现主动场景感知和可控外观操控，是空间增强现实、投影映射和表面反射率获取的核心基础设施。现有的ProCams逆渲染方法能够提供高保真结果，但依赖于耗时的逐场景优化，而主流的前馈3D重建模型生成烘焙外观，无法适应空间变化的投影仪照明。为了解决这一精度-效率权衡问题，我们提出了FF-ProCams，一种面向ProCams的前馈3D高斯逆渲染框架。混合Mamba2-Transformer编码器从稀疏多视角观测中聚合跨视角几何和光度线索，轻量级头部在单次前向传播中预测可重光照的高斯表示。我们进一步设计了投影仪感知的可微渲染器，在任意主动光照和ProCams姿态下合成相机观测。为支持前馈训练，我们构建了大规模合成ProCams数据集，涵盖多样化的物体几何和表面材质。实验表明，FF-ProCams实现了高保真的投影仪感知渲染，泛化到未见模式，并支持新颖的投影仪-相机姿态。仅使用8个输入视角，它便优于使用297个视角的基于优化的基线方法，同时将测试时重建时间降至0.13秒（加速三个到五个数量级）。代码和数据可在https://github.com/CPREgroup/FF-ProCams/获取。

## Abstract
Projector-camera (ProCams) systems achieve active scene perception and controllable appearance manipulation via structured illumination, serving as a core infrastructure for spatial augmented reality, projection mapping, and surface reflectance acquisition. Existing inverse-rendering methods for ProCams deliver high-fidelity results but rely on time-consuming per-scene optimization, while mainstream feed-forward 3D reconstruction models produce baked appearance that cannot adapt to spatially varying projector illumination.   To resolve this accuracy-efficiency trade-off, we propose FF-ProCams, a Feed-Forward 3D Gaussian inverse-rendering framework for ProCams. A hybrid Mamba2-Transformer encoder aggregates cross-view geometric and photometric cues from sparse multi-view observations, and lightweight heads predict a relightable Gaussian representation in a single forward pass. We further design a projector-aware differentiable renderer to synthesize camera observations under arbitrary active illumination and ProCams poses. To enable feed-forward training, we construct a large-scale synthetic ProCams dataset covering diverse object geometries and surface materials. Experiments show FF-ProCams achieves high-fidelity projector-aware rendering, generalizes to unseen patterns, and supports novel projector-camera poses. Using only 8 input views, it outperforms optimization-based baselines with 297 views while reducing test-time reconstruction to 0.13 seconds (a three-to-five-order-of-magnitude speedup). The code and data are available at https://github.com/CPREgroup/FF-ProCams/.

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：提出并构建了**首个针对ProCams系统的前馈3D高斯逆渲染框架**，即**FF-ProCams**。该模型通过直接从稀疏多视角ProCams观测图像中学习，在**单次前向推理**中预测出完整、可重光照的3D高斯场景表示，从而完全避免了测试时的逐场景优化。

- **关键技术细节与算法流程**：
    1.  **网络架构：混合Mamba2-Transformer编码器**：
        - **输入**：稀疏多视角（V=8）的ProCams观测图像，包括黑场图、白场图和主动投影模式图，以及对应的相机位姿。
        - **编码**：将输入图像分块（Patchify）并与相机光线编码（Plücker rays）结合，形成几何感知的令牌序列。
        - **混合序列模型**：采用Mamba2块和Transformer块交替堆叠的混合编码器。Mamba2负责高效的长序列上下文传播，Transformer负责显式建模不同视角间的全局相关性，以聚合跨视角的几何和光度线索。

    2.  **高斯属性预测网络**：
        - 使用两个轻量级线性预测头，分别从编码器输出的令牌中预测几何属性和外观属性。
        - **几何属性**：包括深度`d`、尺度`s`、旋转`q`和透明度`α`。高斯中心`μ`通过深度沿对应相机光线反投影得到，保证了几何一致性。
        - **外观属性**：包括漫反射反照率`a`、表面粗糙度`ρ`和一个辅助环境色`c`。这些属性是物理可分解的，支持后续的投影仪感知渲染。

    3.  **可微的投影仪感知渲染器**：
        - **延迟光栅化**：将预测的3D高斯属性（深度、法线、反照率、粗糙度、环境色）光栅化到目标相机视图上，生成G-buffer图。
        - **投影仪几何映射**：利用光栅化的深度图，将每个相机像素对应的3D点反投影，再投影到目标投影仪的图像平面上，从而获取该点所对应的投影图案像素。
        - **物理着色与合成**：采用简化的Cook-Torrance BRDF模型（包含漫反射项和微表面镜面反射项），结合投影仪光照和环境色，合成最终相机观测图像。其核心公式可概括为：
          - `最终颜色 = 直接投影光照反射 + 环境颜色`
          - 其中直接光照反射通过BRDF和入射光照（从投影仪采样）计算。

    - **损失函数**：
        - **分阶段的光度损失**：按顺序使用黑场、白场、主动投影模式图像进行监督，分别稳定几何、均匀光照响应和复杂光照响应。
        - **掩码熵损失**：约束高斯透明度，使其聚焦于前景物体，抑制背景伪影。
        - **几何正则化**：包括深度损失、深度分布损失和法线平滑损失，确保几何的准确性和平滑性。
        - **粗糙度正则化**：反照率引导的粗糙度平滑损失，防止粗糙度过度拟合照明伪影。

### 8. 不足与局限

1.  **分辨率限制**：由于GPU内存限制，模型训练和推理分辨率限于128x128，这限制了其在高分辨率投影映射应用中的实用性。论文承认这是主要瓶颈。
